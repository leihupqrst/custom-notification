# custom-notification
给通知加自定义View然后定时更新自定义View，如下图所示：  
![image](https://github.com/leihupqrst/custom-notification/blob/master/notification_icon.png)  
使用ArcProgressView显示进度，其中ArcProgressView为自定义View  

步骤：  
1、新建RelativeLayout   
2、把自定义View加进去   
3、View转成Bitmap  
4、给ImageView设置图片SetBitmap()  
5、更新进度
